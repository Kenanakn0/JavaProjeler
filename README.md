# ☕ Java Programming Playground

Merhaba! 👋 Bu depo, Java programlama dilindeki öğrenme sürecimi, nesne yönelimli programlama (OOP) pratiklerimi ve temel algoritma çalışmalarımı içeren kişisel bir kod arşividir.

## 📌 Depo İçeriği
Bu kütüphanede yer alan kodlar genellikle temel mantığı kavramak ve Java'nın özelliklerini test etmek amacıyla yazılmış mini uygulamalardan oluşmaktadır:

* **Temel Algoritmalar:** Döngüler, veri yapıları ve koşullu ifadeler kullanılarak çözülmüş pratik problemler.
* **OOP Pratikleri:** Sınıf (Class), nesne (Object), kalıtım (Inheritance) gibi kavramların uygulandığı mini projeler.
* **Konsol Uygulamaları:** Kullanıcı etkileşimli temel Java terminal programları.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Java (JDK)
* **Geliştirme Ortamı:** VS Code
* **Versiyon Kontrol:** Git & GitHub
