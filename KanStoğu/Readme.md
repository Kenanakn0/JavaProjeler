# Kan Bankası Stok Takibi

Java'da HashMap kullanarak yazdığım ufak bir stok takip projesi. 

Dizilerde (Array) veriyi `for` döngüsüyle aramak yerine, kan gruplarını "Key", stok adetlerini "Value" olarak tutarak veriye doğrudan erişim sağlamayı hedefledim. Sisteme kan girişi ve stoktan kan düşme işlemleri yapılabiliyor.
