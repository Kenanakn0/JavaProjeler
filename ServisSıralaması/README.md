# Acil Servis Triyaj Sistemi

Veri yapıları (Data Structures) dersinde öğrendiğim Priority Queue (Öncelikli Kuyruk) yapısını test etmek için yazdığım basit bir acil servis simülasyonu. 

Sistem, hastaları geliş sırasına göre değil, durumlarının aciliyetine (Kırmızı, Sarı, Yeşil alan) göre sıralayarak muayeneye alıyor. Bunun için `Comparable` interface'ini kullanarak kendi sıralama mantığımı yazdım.
